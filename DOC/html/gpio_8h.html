function to initialize GPIO MODES.

Parameters
GPIOx- Port of GPIO
pin- pin number of GPIO
GPIO_MODE- MODE of GPIO :- OUTPUT/INPUT
GPIO_CNF- Configuration of GPIO

This function initialize diffrent GPIO modes as below

1.GPIO_MODE = OUTPUT , GPIO_CNF = PUSHPULL 2.GPIO_MODE = OUTPUT , GPIO_CNF = OPENDRAIN 3.GPIO_MODE = OUTPUT , GPIO_CNF = AF_PUSHPULL 4.GPIO_MODE = INPUT , GPIO_CNF = FLOATING 5.GPIO_MODE = INPUT , GPIO_CNF = PULL_UP 6.GPIO_MODE = INPUT , GPIO_CNF = ANALOG
